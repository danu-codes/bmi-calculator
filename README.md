# 🧮 BMI Calculator App

A simple and responsive **BMI (Body Mass Index) Calculator** built using **React.js**.  
It allows users to enter height and weight to calculate their BMI and display health status.

---

## 🚀 Features

- 📏 Calculate BMI based on height and weight  
- 🎯 Shows BMI value and health status  
- 🎨 Clean green & white UI design  
- 📱 Fully responsive (mobile + desktop)  
- 🔥 Simple and beginner-friendly React project  

---

## 🌐 Live Demo
👉 Try it here: https://danu-codes.github.io/bmi-calculator/

## 🛠️ Tech Stack

- React.js (Vite)
- HTML5
- CSS3 (Flexbox + Responsive Design)
- JavaScript (ES6)
